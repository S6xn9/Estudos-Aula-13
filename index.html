Anotações de Estudo: Links e Âncoras em HTML5

1. Introdução

Os links (ou hiperlinks) são a essência da Web. Em HTML, o elemento <a> cria essas conexões.

2. O Elemento Âncora <a href="">

A tag <a> define um hiperlink com o atributo obrigatório href que aponta para o destino.

Sintaxe: <a href="destino_do_link">Texto Clicável</a>

3. Tipos de Links

3.1 Links Internos

Exemplo: página Sobre Nós

3.2 Links Externos

Exemplo: Google

3.3 Links para E-mail

Exemplo: nosso suporte

3.4 Links para Telefone

Exemplo: (11) 98765-4321

3.5 Links para Download

Exemplo: relatório anual

4. Âncoras e Navegação Interna

Crie um id no elemento de destino e use <a href="#id"> para navegar.

Exemplo: Ir para Introdução Introdução

5. Atributos Importantes

  • target: Define onde abrir (ex: _blank).
  • title: Dica ao passar o mouse.
  • rel: Relação com o destino (nofollow, noopener).

6. Boas Práticas

  • Use textos descritivos.
  • Garanta acessibilidade e contraste.
  • Teste os links regularmente.

Conclusão

Os links são a espinha dorsal da web. Dominar <a> e âncoras internas é fundamental.
